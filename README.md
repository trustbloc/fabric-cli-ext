# fabric-cli-ext
Command-line interface that extends Fabric's CLI.

```
git clone https://github.com/trustbloc/fabric-cli-ext.git

cd fabric-cli-ext

# run linters
make checks

# run unit-test
make unit-test
```

## Build dependencies

* Go `1.12.5`
* Docker `18.09.x` or above
